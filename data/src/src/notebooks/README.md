# Credit Card Fraud Detection

## Objective
Detect fraudulent transactions using machine learning.

## Algorithms Used
- Logistic Regression
- Naive Bayes

## Features
- Time
- Amount
- Transaction Type
- Location

## Workflow
1. Data preprocessing
2. Model training
3. Evaluation

## Metrics
- Accuracy
- Confusion Matrix
- Classification Report

## Conclusion
Logistic Regression and Naive Bayes are effective for fraud detection.
